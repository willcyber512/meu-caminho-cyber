# Introdução à Cibersegurança — Resumo de Estudos

Este repositório contém minhas anotações completas do curso **"Introduction to Cybersecurity" da Codecademy**.

Os objetivos deste projeto são:
- Consolidar o conteúdo estudado;
- Criar uma base sólida para seguir para cursos mais avançados (Fortinet FCF/FCA);
- Manter um portfólio público para trilha SOC.

Estrutura dos módulos:
1. O que é Cibersegurança  
2. História e Padrões  
3. Ameaças Cibernéticas  
4. Criptografia, Autenticação e Autorização  
5. Segurança de Rede  
6. Proteção de Dispositivos Pessoais  
7. Estudos de Caso e Tendências
