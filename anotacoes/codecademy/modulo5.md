Noções Básicas de Segurança de Rede

## O que aprendi
- O que é uma rede e por que ela é essencial para comunicação.
- Diferença entre LAN, CAN e WAN.
- Para que serve o modelo OSI e como ele organiza a rede.
- Diferença entre modelo OSI e modelo TCP/IP.
- Função de protocolos comuns (DNS, HTTP, SSH, FTP, SMTP).
- O que são firewalls e seus principais tipos.
- Conceitos básicos de segurança em redes Wi-Fi.
- Noções iniciais de monitoramento de tráfego com Wireshark.

## Explicações rápidas
- **Rede:** conjunto de dispositivos conectados para troca de dados.
- **LAN / CAN / WAN:** redes locais, de campus e de larga escala.
- **Modelo OSI:** modelo conceitual de 7 camadas para entender comunicações de rede.
- **TCP/IP:** modelo prático usado na Internet (4 camadas).
- **DNS:** converte nomes de domínio em endereços IP.
- **HTTP:** protocolo usado para acessar conteúdo web.
- **Firewall:** filtra pacotes de rede com base em regras.
- **Segmentação de rede:** divide a rede para reduzir riscos.
- **Wireshark:** ferramenta para capturar e analisar tráfego de rede.

## Como isso aparece no mundo real / SOC
- SOC analisa tráfego de rede para detectar atividades suspeitas.
- Firewalls geram alertas ao bloquear conexões não autorizadas.
- DNS e HTTP são monitorados para identificar ataques comuns.
- Wireshark é usado para investigação de incidentes e troubleshooting.
- Redes segmentadas reduzem o impacto de comprometimentos.
